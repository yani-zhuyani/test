💡 稳定性的核心秘诀
两个关键原则，决定 AI 是否能稳定工作

从「理解规则」→「复制代码」

❌ 告诉 AI "按钮是蓝色32px圆角"
✅ 直接提供 <button class="ks-btn-primary">，禁止改动

独立 CSS 基座

AI 只操作 HTML 结构与 class 名，所有视觉规范在 theme.css 中封存，HTML 变动不会破坏视觉一致性
